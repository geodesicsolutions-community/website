The Discount Codes Add On gives you the opportunity to offer a discount for the areas that you charge a fee. As the site admin, you can set up as many discount codes as you like in the system. Only users who have access to a discount code may take advantage of the discount(s) offered.

The admin's discount code form allows entering a name, description, discount percentage, and the code itself. Additionally, you can set each Discount Code to "active" or "inactive". Your sellers will simply input the appropriate Discount Code during the Listing Process in order to take advantage of the discount.
