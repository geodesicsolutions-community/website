The Zip / Postal Code Search Add On is a versatile search tool that allows your customers to process dynamic zip (postal) code searches. It allows your site visitors to search and retrieve results for classified ads or auctions based upon a zip (postal) code and specified search radius that they enter.

The Add On is delivered ready-for-use with Zip/Postal Codes for the following countries:

  • United States
  • Australia
  • Canada
  • Germany
  • United Kingdom


Import Zip Data feature allows you to import zip/postal code data for other countries directly into the software's admin panel. This addon has been configured to work with .csv data that can be purchased and downloaded from the following sources:

  • geodatas.net
  • geopostcodes.com

The Add On installs easily into your GeCore web site and offers the following flexibility:

1. It will appear on your search page as a searchable field.

2. It can be used with the Enterprise Edition's "Zip Filter", which allows your visitors to filter the entire site's listings based upon the Zip / Postal Code they choose. (see below)

3. You can set up your own html driven search forms and have the Zip / Postal Code field as part of your form.
