Developer's Blueprint for Success...

The Example Add On is the skeletal framework of an Add On with no specific function. The Example Add On is commonly used as a template for code modifications and/or developing your own Add On. Because of this framework, Add Ons can be developed by anyone. Also using the Add On framework for altering or adding to the core functionality of the software eliminates the need to edit the core source code. This will allow for custom modifications and the ability to update to later versions of our software with ease.
