Makes Listing Purchases Easier...

Allow users to use a "Token" that will automatically equal the total main cost of any classified ad, auction, or directory listing. It also allows you to set up token “bundles” that the user can purchase. Tokens can also be added to an account automatically when the account is first created (i.e. during registration).

Once the Tokens Add On is installed and set up, the user will be able to see their tokens in the My Account section of GeoCore. If the user has any tokens on his account:

  • It will display the number of tokens in the “my account links” module (the menu on the left when viewing my account areas).
  • On the cart, for any listing that has a fee (not including listing extras), a button will appear titled “Use token”. When clicked, it will subtract the main cost of that listing, not including listing extra costs. So if a classified ad costs $5 to list for 1 month, and $1 for the bolding extra, a token will subtract $5 and the user will only owe $1 for the “bolding” extra.
  • If the user's price plan is configured to allow purchasing tokens, it will add a button to “purchase tokens” within the my account links module. User will be able to use that to purchase sets of tokens at the prices you set in the price plan settings.
