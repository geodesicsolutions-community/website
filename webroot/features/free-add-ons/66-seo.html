The SEO Add On offers you the ability to launch your site with Search Engine Friendly URL's. The Add On installs easily into GeoCore and uses the Apache mod_rewrite to produce the URL's that Search Engines typically interpret as friendly for increased page indexing.

Novice users will find an intuitive Setup Wizard that guides them through the install process and provides them with suggested default settings.

Advanced users will find a wide variety of settings which allow them to customize the URL's of certain pages to display some additional URL content or turn off the URL content altogether. For example, each URL contains a set of "parts" which appear in the URL bar of a browser window as "directories". The "parts" have been given default names by the software. Each part will be labeled as either "required", "custom", or "optional". This gives you the ability to either edit the name of the "part" to your choosing, or turn "off" parts from being displayed in the URL altogether. So, if you don't like what we've named something, in most cases, you can simply change the text to display whatever you want in the URL.

Automatic character replacement settings allow for the actual listing title that the seller enters to be included in the URL.

Lastly, you also have the ability to set the file extension of the URL... i.e. .htm, .html, .php, etc... perfect for Search Engine Indexing!

NOTE: The SEO addon requires use of “Apache rewrite module” or “mod_rewrite”. Without this enabled, and working properly for your server, the SEO addon will not be able to do its job.
