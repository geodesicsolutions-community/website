Classified Specific Features

These features are specific to the classifieds features within any edition of our GeoCore Software that includes classified ads capabilities. These features by default will only be within the GeoCore Classifieds or GeoCore Max versions of the software.

Note that all classifieds are considered listings within the system so in addition to the following feature all listing features will be included within classified listings also. See the all listing types feature page here

Sold Sign
This feature allows classified sellers to leave their listing up on the site getting more attention even though they may have sold the item. If the item sold in the classified is a renewable quantity the seller can turn the sold sign on until quantities or replenished without the need of creating a new classified ad in the system.
Vote/Comment Feature
If other clients like a specific listing they can vote on it and leave comments about it. The votes are accumulated in a like/dislike score and can be displayed directly within the classified ad details page. All comments left can be browsed by clients viewing the listing. You can limit the voting by a few different methods to stop spamming the feature.

Not Quite Classifieds Features...but close enough

These are feature not specific to classifeds but to auctions. But we find many clients using them instead of classifieds. Those clients use them as "buy it now classifieds" or "buy it now items". The options are wide with these features. Since there is no bidding within the following features many consider them classifieds but with buy it now features. You can dream up your own uses. But the features below were attached to the auction feature because they provided the feedback feature, "bid management" so you can keep track of clients that actually buy the items for sale, ability to charge final fees...etc. Just remember since you control the text throughout the system the labels for these features don't need to include the word auctions in them. Label them anyway you like.

So just be aware to get these features you must have the GeoCore Auctions and the GeoCore Max versions of our software

Buy Now Only Auctions
This bidding or "non-bidding" configuration removes bidding from the auction leaving only the "buy now" link within the auction. This feature really only includes the "auction" name because it takes advantage of all other auction features like feedback and buyer to seller gateway features. You have the ability at the admin to force all auctions or those in a specific price plan to be buy now auctions only.
Details for Buy Now Only Listings
Inventory Item Auctions
This feature allows the seller to sell individual items of a quantity to different buyers from the same auction. Like buy it now there is no bidding within this feature yet still takes advantage of other auction features like feeback and buyer to seller gateway. The seller inserts a quantity of the same items they will sell within the listing and buyers can choose up to the quantity left when they click "buy now". If the buyer does not choose the quantity left what they do purchase is removed from the total so another buyer can buy.
