  • Subscriptions - Power your GeoCore web site with subscription billing. This Add On will allow you to generate recurring revenue through subscription based access.

  • Zip/Postal Codes - A versatile search tool that allows your customers to process dynamic zip (postal) code searches in your installation of GeoCore.
  • Tokens (free listings) - You can automatically issue tokens upon user registration, or after registration, on a user by user basis.

  • Google Maps - Offers you the ability to display mapping information using the Google Map API on the listing details page.

Auction Specific Features

  • Standard Auctions - Feature allows sellers to place their individual items up for auction.
  • Dutch Auctions (Multiple Item Auctions) - Feature allows sellers to post several identical items or "lots" of items up for auction. Bidders will then bid a certain amount per item on a selected number of those items. This type of auction has the potential of having several winning bidders on the same auction...each bidder winning a certain number of items.
  • Reverse Auctions - This auction type is virtually the same as the standard auction except that bidding goes lower instead of higher or more specifically the roles of the buyer and seller are reversed. In this auction type sellers compete to obtain business.
  • Minimum Bid - This feature allows the seller to set the minimum starting bid for the auction.
  • Reserve Bid - This feature allows the seller to specify the minimum amount they are willing to accept for their auction from a bidder.
  • "Buy Now" Auction - This feature allows willing bidders to purchase the auction item immediately at the seller's specified "Buy Now" price.
  • Proxy Bidding - Sometimes called "auto-bidding" or "absentee bidding", this feature allows the bidder to enter the maximum amount that they are willing to pay for the item and then walk away. The software will then automatically bid on their behalf up to the amount they specified...absentee bidding!
  • User/Seller Feedback Rating System - This feature allows the seller and winning bidder of each auction to rate each other and provide a comment on how well the transaction went. The system uses "icons" that represent a user's favorable or 'not so' favorable feedback ratings. For instance, by default the system uses colored stars to represent each user's feedback. You can easily replace the default star images with your own if you like.
  • Final Fees - Upon successful completion of an auction, you can charge the 'seller' a percentage fee based upon the 'final' selling price of the item.
  • Bid Status - Bidders can view the current status of their bids by logging in to their personal "My Account" pages.
  • Seller Emails - The software will send an email to the seller when their listing has been successfully placed on your site. Additionally, the seller will receive emails when their auction is about to expire, when the auction has been "unsuccessful", and when the auction has received a winning bidder.
  • Bidder Emails - The software will send emails to the bidder when they have submitted a successful bid, when they have been outbid, and when they are the winning bidder of an auction.
  • Bid History - You can allow visitors to view the current bid history of an auction. Bid history displays the bidder's username and bid amount.
  • Invited List of Buyers/Bidders - Each user can set up a list of buyers (based upon their usernames) that they are willing to accept bids from on their auctions. Only those users on the "Invited List" will be allowed to place bids on the user's auctions.
  • Black List of Buyers/Bidders - Each user can set up a list of buyers (based upon their usernames) that they want to "black list" from bidding on their auctions. Only users on the "Black List" will not be allowed to place bids on the user's auctions. All other users on the site can place bids on the user's auctions.
  • Auction Start / End Time - You can give the seller the ability to set a specific date and time for each auction to begin and end, rather than the auction simply going live immediately after placing the auction.
  • Auction Preview Period - If allowing your sellers to specify the Auction Start / End Time, you can also set your site so that these auctions can be previewed on your site. All of the Listing details will be available to visitors of your site. However, they will not be able to place bids on the items until the seller's specified Start Time arrives.
  • Subscription Based Bidding - You can set up your site so that each user must have a subscription to place a bid and/or even view an auction.
  • Endless Auctions (Sniper bidding) - This feature automatically extends the length of an auction if a user bids just before the auction was to end. You can specify the number of minutes prior to expiration that the system should check for a successful bid. If a successful bid is detected within that time, the auction will be extended for the number of minutes you specify. This process will continue over and over until all bidding has stopped.
  • Seller to Buyer Gateway (PayPal) - Sellers can accept payments from buyers (winning bidders) for successful auctions using their PayPal accounts. Seller can enter their PayPal id into their account which will indicate that the seller accepts this form of payment from buyers.

  • YouTube Video Uploads - Charge users for posting YouTube Videos on your site or set to 'free'
  • Payment Methods - You can can accept payments from your users in the following ways
    • Real-Time Credit Card Processing - Allow users to pay you for their listings by credit card. The listing can be set to automatically go 'live' when the payment is processed.
    • PayPal Payments - Allow users to pay you for their listings through their PayPal account. The listing can be set to automatically go 'live' when the payment is processed.
    • Cash Payments - Allow users to pay you for their listings by cash. The listing will be placed on hold until you receive their payment.
    • Check Payments - Allow users to pay you for their listings by check. The listing will be placed on hold until you receive their payment.
    • Money Order Payments - Allow users to pay you for their listings by money order. The listing will be placed on hold until you receive their payment.
    • Recurring Billing - Automatic recurring billing for subscriptions. Requires the Subscriptions Add On.
  • Listing Fees - can be charged based upon the following criteria:
    • Flat Fee - for each listing.
    • Price Field - of the item.
    • Duration (in days) - of the listing.
  • "Listing Extras" Fees - You can charge users specific fees for selecting extras to be added to their listings:
    • Attention Getters
    • Featured Listing Status
    • Better Placement Status
    • Bolding Status
    • Fee Per Photo
    • Fee Per YouTube Video
  • Renewal Fees - Charge users for renewing their listings.
  • Taxes - Charge your users a tax per listing based upon their location. The States/Provinces/Countries you have set up in the Geography Setup Menu allow you to specify a tax per region. Additionally, you may charge a 'flat tax' or a 'percentage tax' based upon the total of the purchase (listing cost).
  • Cart System - Allows your sellers to walk through a 'shopping cart' style process for placing listings, etc. on your your site. They can place multiple listings at the same time and pay for them once. Items will remain in their cart for a period of time even if they leave the site and return at a later time.
  • Multiple Listing Process - Allows your customers the ability to purchase multiple listings/upgrades/renewals/subscriptions in the same session with a "single checkout process". This feature saves your customers time from having to submit their payment details over and over each time they list an item on your site. It also saves you money because it reduces the number of merchant account transaction fees.
  • User Group Specific Pricing - Charge different fees to different user groups. You can separate users into groups when they register so that you can charge them differently. Requires the Enterprise Pricing Add On.
  • Unlimited Price Plans - You will have the ability to offer as many different price plans as you like... fee-based or subscription-based. Requires the Enterprise Pricing Add On.
  • Category Specific Pricing - Charge certain fees for listings placed in certain categories.
  • Price Plan Expiration - Price Plans can be set to expire so that you can offer promotional specials to users for a period of time before an expired plan is replaced by another plan. Requires the Enterprise Pricing Add On.
  • Subscriptions - Charge users to have a subscription to your site. They can then place as many listings as you allow them to, but still be required to pay you for the listing "Extras" such as Attention Getters, Featured Status, etc. Requires the Subscriptions Add On
  • Account Balance System - Allows your users to have a running 'account balance' with your site, from which they can place, renew, and upgrade listings. Users can add money to their balance any time they wish through a link within their "My Account" section of your site. You can also allow them to have a 'negative balance' and be billed for their transactions later. Requires the Account Balance Add On.
  • Featured Listing Levels (5 levels) - Lets you charge based upon location (higher exposure) of the listing on your site. For example, if you were to place Featured Level 2 listings on the Front Page, you would charge users who select this option more than you would charge users for selecting Featured Level 3 listings that you place on a subcategory page. Charge more for better exposure. Requires the Featured Level Add On.
  • Discount Codes - Ability to give users a special discount code that will give them a percentage discount off of the total listing cost during the listing process. You can create and issue as many discount codes as you like. Discount codes can also be applied towards 'recurring billing'.
  • Subscription Required to View Listings Option - This setting allows users to browse your site's categories but requires them to pay for a subscription in order to view listing details pages. Requires the Subscriptions Add On.
  • User Selected Price Plans - Based upon your permission settings, users can select their own pricing scheme when they go to place a listing on your site. Requires the Enterprise Pricing Add On
  • Storefront Add On - Offers each seller who has a StoreFront subscription an interactive set of pages built directly into your site. This gives the seller the opportunity to highlight their listings.

User Registration & Login Controls

  • Registration Page - Simple, one-page registration process.
  • Disable Registration - Ability to turn off ability for users to register on the site.
  • Anonymous Listing Process - Offers you the ability to allow for classified ad placements on your site without requiring the seller to first sign in, or even register at all for that matter.
  • Just-In-Time (JIT) Registration Process - This is an alternative to the standard Registration Page. It allows your sellers to complete most of the listing process prior to asking them to register or log in. This is a very convenient way to get sellers to post listings on your site.
  • Secure Login - Secure User Login through ability to set SSL when logging in.
  • Email Verification Routine - You can implement an email address verification routine that forces the user to confirm their email address:
    • Reduces fake registrations
    • Ensures unique email addresses are required (no duplicates within the site)
    • Assures that the user registering at least has current access to the email address used
  • Lost Password Routine - Sends the user's password to their email address.
  • Registration Fields - You can specify which fields are to be used and required. If the user fails to fill in a field, an error message displays.
  • Suspend Users - Admin has the ability to suspend users.
  • Maximum Listings - Admin can set the maximum number of listings a user can have active at any given time
  • Forum Bridge Add On Available - Allows you to share user registrations across multiple installations of our software or a Bulletin Board.
  • IP Banning - You have the ability to block registrations by users from specified IP Address.
  • Post Login Page - You can specify that either the Front Page or My Account Page is displayed after a user logs in.
  • Affiliate System - Software is compatible with your license of either of the following Affiliate Software programs, which allow affiliate sites (customers) to refer new customers to your site and earn percentage based commissions on the listing placements that those referred customers purchase through your site:
    • iDevAffiliate
    • Post Affiliate Pro
  • Security Image - A Security Image will display at the bottom of each form for your visitors to verify each time they submit a form.
  • Registration Code - You can create User Groups and assign each a unique Registration Code. When a user registers with that code, they will automatically be placed into that User Group and subject to that User Group's Price Plan fees. This allows you to advertise radio spots to some groups with one code, newspaper spots with another code, and etc.
  • Tokens (free listings) - You can automatically issue tokens upon user registration, or after registration, on a user by user basis.
  • Registration Splash Page - When a user enters a Registration Code during registration you can display a Splash Page to them which displays any html you wish. For example, you might use the splash page to review the advertised pricing scheme that they will enjoy while using your site.
  • Custom Registration Fields - Add up to 10 custom registration fields and set them to 'required' or 'dependent' based upon the use of another Registration Field.
  • Domain Blocking - You have the ability to block registrations by users from specified domains
  • Domain Allowing - You have the ability to allow registrations only by users from specified domains.
  • Starting Balance - You can automatically issue credit (money) into a user's account balance upon registration. Requires the Account Balance Add On.
  • Registration Required to Contact Seller Option - This setting allows visitors to browse your site's categories and listings but requires them to be logged in to contact the seller.
  • Hide Fields - This feature allows you to specify fields on the classified detail or auction detail display page that will be hidden from view if your visitor is not currently logged in. Your visitors will be enticed to register in order to view that information.
  • Membership Feature - "Registration Required to View Listings" setting allows visitors to browse your site's categories but requires them to login/register in order to view individual listings. This allows you to charge them after the registration process for a 'subscription/membership' to your site. Requires the Subscriptions Add On.
  • Admin Login as User - Admin has the ability to login to the customer side of the software as if they were the user, by using their admin password.

My Account Pages

  • My Account Pages - Each registered user will have their own personal My Account section on our site in which they can edit their registration information and personal settings.
  • Cart System - This system guides the seller through all of the processes regarding listings, renewals, subscriptions, StoreFront purchases, etc. It even remembers the last process that the seller was performing if the seller got distracted doing something else.
  • Listing Process - Process guides the user through the Listing Process, which consists of choosing a category/subcategory for their listing, adding photo(s), entering the listing's details, and the payment processing for the listing.
  • My Active Listings - This section displays all of the user's active listings on your site. Based upon your permission settings, the user can edit and renew their listings prior to its expiration.
  • My Expired Listings - This section displays all of the user's expired listings on your site. Based upon your permission settings, the user can renew their listings within a set number of days you specify after expiration.
  • My Favorites - Allows the user to add listings on your site to a 'favorites' or 'watch list'. These listings may be either removed by the user or automatically when the listing expires.
  • My Listing Filters - Allows the user to set up 'filters' or 'email notifications' in which the software will monitor user specified categories and key words on the user's behalf, and automatically notify the user by email if a listing fits their criteria. This routine is run each time a listing is added to the site.
  • My Account Information - Allows the user access to their personal (registration) information. They have the ability to view and edit their data, as well as, specify which of these fields they are willing to display publicly on their 'sellers other listings' page. A table of current fees is also displayed to them on this page.
  • My Messages - Allows the user to communicate with other users of the site through a personal messaging system. They can view and reply to messages that were initiated through the 'contact seller' link on the Listing Display Page. Buyers can ask questions to the seller using the "Contact Seller Form" and request (by clicking a checkbox) to have their question be made public. If the seller answers their question and agrees (by clicking a checkbox in their "My Communications" section) then that Question/Answer will be displayed on the listing display page for others to view.
  • Message Settings - Allows the user to specify either "Public" or "Private" communications when using the software's messaging system. The "Public" setting will reveal the user's email address. The "Private" setting will keep the user's email address private.
  • Renew Listing - User can renew their listing prior to the listing's expiration, or within so many days of expiration, per your permission settings.
  • Copy Listing - Sellers can create a new listing by copying one of their existing listings. This saves them time from having to type duplicate information.
  • Upgrade Listing - User can upgrade their listing during the renewal process by adding new feature 'add ons', such as featured status, bolding, and better-placement.
  • Account Balances - If you allow for 'account balances' to be used on your site during the listing process, the user will be able to view their account balance, add money to their account balance, and view past transactions on your site. Requires the Account Balance Add On.
  • Signs & Flyers - Each user has the ability to print out a sign and flyer for each of their active listings on your site. Each sign and flyer is automatically populated with the listing information they have already entered during the listing process. You will predefine the look of the sign and flyer by implementing a template with the desired html and fields. When creating the template, the user can edit the fields prior to printing if they wish to.
  • My Current Bids - Shows the bidder their current and past bids on the site, as well as, the status of those bids...i.e. 'outbid', 'high bidder', etc.
  • Feedback Management - Ability for users to view and provide feedback for auctions that they have either placed or been the high bidder on. Feedback comments and the seller's rating are displayed on each of the seller's Listing Display Pages.
  • Invited List of Buyers - Each user can set up a list of buyers (based upon their usernames) that they are willing to accept bids from on their auctions. Only those users on the "Invited List" will be allowed to place bids on the user's auctions.
  • Black List of Buyers - Each user can set up a list of buyers (based upon their usernames) that they want to "black list" from bidding on their auctions. Only users on the "Black List" will not be allowed to place bids on the user's auctions. All other users on the site can place bids on the user's auctions.

Searching & Browsing Capabilities

  • Advanced (Dynamic) Search Page - that allows for a variety of different search criteria. When the visitor selects a category to search, the search page refreshes and presents to the visitor additional "fields" that relate to the "category" they just chose. All category specific fields that the "seller" was given during the listing process are also "searchable fields" for the visitor to use on the Advanced Search Page.
  • Category / Subcategory Browsing - Visitors will find your site easy to navigate as they click from one category to the next and through your site's subcategories.
  • Zip / Postal Code Search Add On Compatible - Allows visitors to enter their U.S. Zip Code and a search radius from that Zip Code. The software will return a set of results that meet the specified "proximity" search radius. Requires the Zip / Postal Code Add On.
  • Featured Listings - Displays a list of your site's featured (high profile) listings.
  • Newest Listings - Displays a list of the listings most recently placed on your site.
  • Hottest Listings - Displays a list of your site's most popular listings.
  • Search Box - Displayed on front page and/or subpages of your site and allows for search by keyword and category.
  • Sortable Listings - While browsing categories, visitors can 'sort' the listings by 'title', 'description', 'city', etc. Any "Extra Fields" that you are using for each category are also 'sortable' by the visitor.
  • Recent Listings Links - Allows your visitors to view only those listings that were placed within the last 24 hrs, one week, two weeks, or three weeks.
  • Custom Search Forms - You have the ability to create your own html search forms to be displayed on most pages of your site. Simply create the form using html and use the software's normal Search routine to produce results.
  • Geographic Navigation - Offers you the ability to offer our visitors browsing by country, state/province, or other "geographic region" criteria that you set using unlimited region levels.
  • Listing Tags (with auto-complete!) - Allows seller to specify "tags" to be set for each listing. A listing tag is basically a keyword that applies to the listing:
    • Browsing - The user can browse the site by listing tag, similar to browsing by category.
    • Auto-complete - Auto-complete feature for listing tag field in listing placement and listing edit process.
    • SEO - Browse by tag uses SEO re-written URL once configured.
    • Searching - Can search by tag, with auto-complete built into the search.
  • Contact Us Form - Site visitors can contact you through this secure form embedded into your site. Set up multiple contact emails and departments.
  • Custom Search Page - You have the ability to deactivate the default Advanced Search Page and insert your own html driven search form instead. Additionally, you can set up your html driven Search Page for each category of your site.
  • State Filter - Allows visitors to filter out listings by selecting their state from a dropdown menu.
  • Zip (Postal) Code Filter - Allows visitors to filter out listings by selecting a zip (postal) code from a dropdown menu. This filter uses a proximity search (radius search) routine. Requires the Zip / Postal Code Add On.
  • Field Filter - Allows you to set up two or more levels of dynamic dropdowns for your visitors to select from. These dropdowns, when selected by the visitor, will automatically filter listings on the site based upon the value chosen by the visitor.

Security

  • Email Notification - Sent to the administrator when a user registers.
  • Badword Filter - Screen listings as they are entered onto your site for "badwords". Designate an automatic 'replacement word' for each 'badword' that the software detects.
  • Email Validation - Checks the validity of e-mail addresses to.
  • Form Field Validation - Checks to make sure that users entered all necessary contact information.
  • Register Globals - Code written to allow for register globals "on" or "off" without sacrificing security.
  • Secure Registration - Ability to set the registration process to secure mode by using your domain's security certificate.
  • Secure Login - Secure User Login through ability to set SSL when logging in.
  • Secure My Account Area - Secure My Account browsing through ability to set SSL when logging in.
  • Secure Listing Process - Ability to set the listing process to secure mode by using your domain's security certificate.
  • Duplicate Email Checking - System checks existing registrations to ensure that the email address entered is not already in use.
  • Duplicate Username Checking - System checks existing registrations to ensure that the email address entered is not already in use.
  • Username and Password Checking - System checks for to ensure that user has entered the minimum and maximum number of digits for their username and password.
  • Validate Password Routine - System requires the user to enter their password twice to ensure accuracy of the entry.
  • HTML Restriction - Declare specific HTML tags that will be allowed/disallowed on your site.
  • IP Banning - Ban users from accessing your site who are connecting from a specific IP Address. Users from these IP Addresses will not even be able to view your site at all.
  • Notices and Warnings - Section in admin which provides warnings to you of potential security issues.
  • Security Image (CAPTCHA and reCAPTCHA) - User required to enter security code when filling out registration, login, contact seller, and notify-a-friend forms on your site. Code is produced by a random generator and presented to the user to enter for validation.
  • Debug Logging and Messaging - Logs debug output for your site, which is helpful for third party development or if your site is experiencing problems.
  • Domain Blocking - Admin has the ability to "block" registrations by users from specified domains.
  • Domain Allowing - Admin has the ability to "allow" only registrations by users from specified domains.
  • Listing Edit Email - Sent to the administrator when a seller edits their listing.
